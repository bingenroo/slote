# Draw example

Example app for the `draw` component.

## Purpose

Run and debug drawing without the full Slote app:

- Pen, highlighter, eraser
- Color and stroke width
- Canvas interaction
- Drawing vs view mode

## Run

```bash
cd components/draw/example
flutter pub get
flutter run
```

## Features

- **Pen / highlighter / eraser**: Tool selection
- **Color picker**: Predefined colors
- **Stroke width**: 1px–20px
- **Clear canvas** and drawing mode toggle
- **Info bar**: Stroke count and tool

## Tips

1. Try different stroke widths and colors
2. Switch tools while drawing
3. Test eraser and highlighter
4. Toggle drawing vs view mode
