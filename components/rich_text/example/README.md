# Rich text example

Example app for the `rich_text` component.

## Purpose

Run and debug rich text editing without the full Slote app:

- Text input and editing
- Format toolbar (bold, italic, underline)
- Text selection
- Character/word stats

## Run

```bash
cd components/rich_text/example
flutter pub get
flutter run
```

## Features

- **RichTextEditor**: Multi-line input
- **FormatToolbar**: Bold, italic, underline
- **Clear** and text statistics

## Tips

1. Select text and apply formats
2. Combine formats (e.g. bold + italic)
3. Use the stats bar for character/word count
