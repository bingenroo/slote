# Licensing & Attribution (rich_text)

This repository’s `components/rich_text` package integrates [`appflowy_editor`](https://pub.dev/packages/appflowy_editor) to provide the Markdown import/export editor experience (including tables).

## License choice

`appflowy_editor` is dual-licensed under **MPL-2.0** and **AGPL-3.0**. For this project we operate under **MPL-2.0**.

## Attribution requirements (MPL-2.0)

When distributing Slote, include an attribution notice that Slote uses `appflowy_editor` under **MPL-2.0**, and provide a way for recipients to obtain the corresponding source code.

## Web checklist (important)

Because the editor runs on **Web** (Flutter Web), ensure the Slote app’s About/Licenses UI includes the `appflowy_editor` MPL-2.0 attribution notice (and links to the relevant source), so your distribution obligations are met.
